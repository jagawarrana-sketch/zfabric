❓ Help — How to Use
Zayyan Fabrics ERP — Quick Guide
🔍 Search (Ctrl+K)
Koi bhi account, item ya voucher dhundne ke liye search bar use karein. Ctrl+K se search bar focus hoga.
📊 Data Analysis
Charts aur graphs se apni business ki performance dekhen. Sales trend, purchase trend, top parties.
📋 Party Ledger
Kisi bhi form mein "Open Ledger" button click karein. Party ki puri history debit/credit ke saath dekhein.
🖨 Print Options
Har form mein "Print ▾" button hai. Multiple print options milte hain jaise voucher, challan, parchi.
💾 Data Save karna
Har form mein "Save" button hai. Deploy hone ke baad data database mein save hoga permanently.
➕ New Account
Purchase, Sale ya Contract mein "New Account" button se seedha account create kar sakte hain.
